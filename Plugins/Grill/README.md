# Grill

<br>

Grill is an AI-powered chatbot with one job: making fun of you. Start a conversation, ask it anything, or let it comment on your coding habits. Expect sarcasm, questionable life advice, and the occasional personal attack.

Don't take it too seriously.

<br>

## Features

<br>

- Chat with Grill
- Random, unprompted roasts
- Sarcastic and humorous responses
- Context-aware conversations
