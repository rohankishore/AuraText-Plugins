# Live Server

Start a local development server directly from Aura Text and preview your web projects in your browser. Changes are reflected automatically as you edit your files, making front-end development faster and more convenient.

## Features

- Start and stop a local development server
- Automatic browser reload when files change
- Supports HTML, CSS, and JavaScript projects
- Serve any project folder
- Lightweight and easy to use

## Getting Started

1. Open a web project in Aura Text.
2. Run **Go Live** from the Status Bar / Command Palette.
3. Your default browser will open with the local server.
4. Edit your files and refreshes will happen automatically.

To stop the server, click **Go Live** again.

## Requirements

No additional configuration is required for most projects.
